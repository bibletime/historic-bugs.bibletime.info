Description Damian Wrzalski 2013-12-24 15:12:19 EET 
I've do a clean installation of BibleTime 2.10.0 (private ebuild). After change any settings from Desk -> Text filters and click Apply/OK there are no changes in settings. After reopen this settings, it's still unchanged. Other setting (Standard works for example) works as expected.
