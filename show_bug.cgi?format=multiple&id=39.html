Description yvand 2014-05-06 20:06:59 EEST 
I work on FreJND module.
BibleTime does not always break paragraphs. It appears BibleTime does not break a paragraph if it is in the verse (not at the end as usual).

We use osis markup. The OSIS file is available at http://koocotte.org/darby/darby.osis.zip, whereas a zipped version of the module at http://upload.legtux.org/get.php?id=5955469201
The problem occurs for instance at 2Sam.2.4.

It works fine with Xiphos and `diatheke -b FreJND -k 2Sam.2.4` gives:
« Et les hommes de Juda vinrent et oignirent là David pour roi sur la maison de Juda.  <div eID="gen1906" type="x-p"/> <div sID="gen1907" type="x-p"/> Et on rapporta à David, disant : Ce sont les hommes de Jabès de Galaad qui ont enterré Saül. » (as you can seen there is a paragraph break in the middle of this verse).
So I guess this is a BibleTime bug.
