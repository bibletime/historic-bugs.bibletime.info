The texts to translate for the final 2.10 release will (hopefully) be fixed when we release 2.10 Release Candidate 1. We call this a "string freeze". After a string freeze, no new strings will be added, so that translators can update the translations without worrying that the texts change before the final 2.10 release. We will notify translators primarily via the translators mailing list.

We have not yet fixed a schedule for the release of 2.10. Once we do, we can also notify the translators.
