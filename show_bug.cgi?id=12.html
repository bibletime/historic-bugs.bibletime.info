Description Jaak Ristioja 2012-03-10 17:56:41 EET 
Currently the "Analyse result" window only shows number of occurances per book. It should be possible to have more detailed analysis, e.g. occurances per chapter etc.

This bug was moved from the old tracker: https://sourceforge.net/tracker/index.php?func=detail&aid=3501072&group_id=954&atid=350954

The original reporter, Christopher Vella, noted that it were nice to have a detailed analysis feature alike the software "Biblie Clerus" has. If anybody has a copy of that software it might help to have a more detailed description or screenshots about how they do it.
